What is a web developer?

Aaaaa web developer is a programmer who specializes in, or is specifically engaged in, the development of World Wide Web applications using a client–server model. The applications typically use HTML, CSS and JavaScript in the client, PHP, ASP.NET (C#), Python, Node.js, Go or Java in the server, and http for communications between client and server. A web content management system is often used to develop and maintain web applications.

All about Back-End Developers

A back-end developer is someone who builds and maintains the technology needed to power the components which enable the user-facing side of the website to exist. Their back end code adds utility to everything the front-end developer creates.
Common task:

· Database creation, integration, and management
· Back-end frameworks to build server-side software
· Web server technologies
· Cloud computing integration
· Server-side programming languages
· Operating systems

    Full stack?
    Facts and truth

    #1 21 October 2021

    How high full-stack salary compare to front and back?

    - According to estimates from Glassdoor, Indeed, and ZipRecruiter, the average full stack web developer salary is between $105,813 and $108,984 per year. These numbers are based on the US in 2020, and don't include bonuses, stock options, and other compensation benefits.
    - In Vietnam, a programmer's salary can reach more than 1,400 USD. In the trend of industry 4.0 with a series of new technologies, programmers are not difficult to achieve a salary of over 1,000 USD.
    - To compare the salary between frontend, backend and full stack, the information in indeed.com will be used (US, October 2021):
    • The average Front-End Developer salary annually is $102,453
    • The average Back End Developer salary annually is $115,344
    • The average Full Stack Developer salary annually is $106,177

    The popularity of full stack?

    - According to a 2020 Stack Overflow survey of 65,000 developers worldwide, roughly 54.9 percent, identify as full stack. 55.2 percent of those surveyed identified themselves as back end developers, 37.1 percent self-identified as front end developers, and just 19.2 percent claimed to be mobile developers.

    #2 21 October 2021

    What is full stack development?

    - Full stack development is the process of working on a project single-handed, which is to do not only the client side, but also the server/backend side.

    #3 21 October 2021

    Why do we need a full stack, but not a pair of front and back?

    - How can a single full stack developer be better than a pair of front / back developers? These are the advantages for most of the time:

    • Master all the techniques involved in a development project, understand all aspects of new and upcoming technologies, therefore can reduce the need and cost of people involved the project
    • Complete the prototypes very rapidly, more time for team communication
    • You can switch between front and back end development based on requirements, and can help other members of the team

    #4 24 September 2021

    What does a person need to learn to be a full-stack developer? Front + end knowledge?

    - First, you have to know about the details of both the jobs:

    • Front End Developers:
    *Deal with the interface and user-facing systems
    *Develop the navigation, interface, and visuals
    *Focus on user experience
    *Utilize HTML, CSS3, and JavaScript

    • Back End Developers:
    *Deal with the server and database of the application
    *Create the back frame that allows the front end development to exist
    *Utilize Python, Ruby, and PHP
    *Store user information into database

    - So, to be able to work as a full stack developer, you have to know and perform well at most of these specified requirements. That says you do not have to be an expert at all of these things.
